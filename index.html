Badges for GitHub and Bitbucket README.md files

Markdown code for lots of small badges (from shields.io, forthebadge.com etc)

Download this project as a .zip file Download this project as a tar.gz file

List of Badges

A list of badges, with their Markdown code, that can be included in a README.md file for a GitHub or Bitbucket project.
